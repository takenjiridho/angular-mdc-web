Linear Progress

Progress indicators express an unspecified wait time or display the length of a process.

Design & API Documentation

Selector: mdc-linear-progress

Properties Description
open: boolean Set the linear progress indicator to an opened or closed state.
determinate: boolean Puts the linear progress indicator in an determinate or indeterminate state.
reversed: boolean Reverses the direction of the linear progress indicator.
secondary: boolean Colors the button with the secondary theme color.
progress: number Sets the progress bar to this value. Value should be between [0, 1].
buffer: number Sets the buffer bar to this value. Value should be between [0, 1].
